# Valentine's Day Website

A minimalist, interactive Valentine's Day website created to test Claude's web development capabilities.

## About

This project was created as a test to explore Claude's ability to:
- Design and build responsive web interfaces
- Create interactive user experiences
- Implement smooth animations and transitions
- Develop mobile-friendly applications

## Features

- 3 interactive pages with smooth transitions
- Fully responsive design (works on mobile and desktop)
- Minimalist aesthetic with gradient backgrounds
- Animated elements and floating hearts
- Pure HTML/CSS/JavaScript (no frameworks required)
